# Jogo War

### Instalação
```bash
# Crie um ambiente virtual com Python 3.12
conda create -n jogoWar python=3.12 -y
conda activate jogoWar

# Instale as dependências do projeto
pip install -r requirements.txt
```

### Iniciando o servidor
#### 1. Back-end
